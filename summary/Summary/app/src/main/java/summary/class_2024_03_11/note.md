### Spring IoC 컨테이너 구동원리
- XML, .class에 등록된 빈은 여러 작업을 통해 객체로 생성되는데
  - 해당 작업을 수행할 객체가 존재한다면 Spring IoC 컨테이너는 그 객체를 사용해 작업을 수행한다.
  - 객체가 없다면 실행하지 않는다.
  - ex) 닭 -> 털뽑은 닭 -> 튀김옷 -> 양념하는 객체가 없다면 -> 후라이드 치킨(객체) 생성
